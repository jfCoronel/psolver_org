Welcome to pSolver

pSolver is a free, on-line tool designed to analyse, solve, edit and manage problems. The development of the application attempts to circumvent the main disadvantages of using "equation solvers" for learning scientific and technical subjects. The application allows us to pose and solve problems in the same way as we would do it on paper, facilitating their review, evaluation, management and editing by teachers and students. In addition to data, equations and unknowns, the application allows the addition of enriched texts (Markdown), images, fluid properties (CoolProp), tables and graphs.

pSolver

The development of the application started in 2013 under the name "CuadernoDeProblemas", in 2023 it changed its name to "pSolver". The main modifications it has undergone can be found in the history.

pSolver is an on-line application to facilitate the resolution of problems that usually arise in the technical disciplines related to engineering. The following video shows a tutorial with a very simple geometric problem (Spanish introduction).

In this page you can find a user manual where each of the features of the tool and how to use them are described.

In the articles and videos section we will include documentation and tutorials to facilitate the use. We also want to ask for your collaboration for improvement and debugging, we have created a user group so that you can inform us of bugs, improvements or opinions.

Recommended browser

It is recommended to use Chrome, Safari or Microsoft Edge. With Firefox it is not allowed to use the system clipboard to copy and paste components, we can enable it by following the steps below:

  1. Type "about:config" in the Firefox navigation bar.
  2. Accept the "Accept risk and continue" message.
  3. Search for "dom.events.testing.asyncClipboard" and set it to "true".

Problems are automatically stored in an on-line database (Google Firebase). It is possible to use the web application in off-line mode (without internet connection). Changes to problems are saved locally and synchronized when the same browser is reconnected to internet.
